# 仮説: 高価格帯における材料品質フィルタリング効果

**作成日**: 2025-11-20
**ステータス**: 仮説（検証済み・データ裏付けあり）
**分類**: v2.1改善候補

---

## 仮説の内容

**6000-10000円の高価格帯は、デイトレーダーから資金的に敬遠されるため、Grokで選ばれる際には強い材料（決算修正、IR、適時開示など）が必要となり、結果として材料の質が高く、勝率・リターンが良くなる。**

---

## 背景

買いシグナルの勝率が39.0%と低い中、株価帯別に分析したところ：
- **6000-10000円: 50.0%勝率、+0.69%平均リターン**（唯一のプラス）
- 他の価格帯: 全てマイナスリターン

さらに、6000-10000円は：
- 勝った時の平均: +3.05%
- 負けた時の平均: -1.67%
- **勝ち負け比率: 1.83倍**（全価格帯で最高）

→ ユーザーの体感「6000円以上は勝率が高い気がする」を検証

---

## 仮説のメカニズム

### 1. 低価格帯（0-3000円）の選定プロセス

- **参入障壁: 低い**
  - 100株購入: 0-30万円
  - デイトレーダーが殺到しやすい

- **Grok選定基準:**
  - SNSバズ（「ストップ高狙い」「急騰銘柄」投稿）
  - 出来高急増
  - テーマ性連動

- **材料の質:**
  - 「Xで投稿急増」「バズ」中心
  - ファンダメンタルズが弱くても選ばれる
  - **ノイズ・投機的要素が多い**

### 2. 高価格帯（6000-10000円）の選定プロセス

- **参入障壁: 高い**
  - 100株購入: 60-100万円
  - 資金的に参加できるデイトレーダーが限定される

- **Grok選定基準:**
  - **決算上方修正**
  - **IR発表**
  - **適時開示**
  - ファンダメンタルズ材料中心

- **材料の質:**
  - 具体的な業績改善（経常利益43.2%増など）
  - 公式発表（適時開示、IR）
  - **SNSバズのみでは選ばれない**
  - **材料の純度が高い**

### 3. 結果

高価格帯では：
- **Grokが暗黙的に材料品質をフィルタリング**
- 弱い材料の銘柄は排除される
- 強い材料の銘柄のみが選ばれる
- → 勝率・リターンが向上

## v2.1への示唆

### 単純な価格帯加点は不十分

❌ **悪い実装:**
```python
if 6000 <= current_price < 10000:
    score += 20
```

理由: 価格だけでは因果関係を捉えていない

### 推奨実装: 価格帯 × 材料品質

✅ **良い実装:**
```python
# 高価格帯 × ファンダメンタルズ材料
if 6000 <= current_price < 10000:
    if has_earnings_revision or has_ir_announcement:
        score += 30
        reasons.append("高価格帯 × 強材料（決算修正/IR）")
    elif has_fundamental_catalyst:
        score += 15
        reasons.append("高価格帯 × ファンダ材料")
    else:
        # SNSバズのみの場合は加点しない
        pass
```

### 必要なデータ追加

v2.1で以下を判定できるようにする：
- [ ] 決算修正の有無（J-Quants API）
- [ ] IR発表の有無（Grok reasonから抽出）
- [ ] 適時開示の有無（Grok reasonから抽出）
- [ ] SNSバズのみか判定（reason文字列解析）

## 残課題

### 1. サンプル数が少ない
- 6000-10000円: わずか6件
- 統計的有意性の検証が必要
- → 今後のデータ蓄積で再検証

### 2. 材料品質の自動判定
- 現状: Grok reasonの文字列解析のみ
- 必要: J-Quants APIから決算修正を取得
- 必要: 適時開示データベース連携

### 3. 10000円以上の扱い
- 10000円以上: 勝率16.7%（非常に悪い）
- 超高額株は別ロジックが必要
- → v2.1では10000円以上は**減点**すべきか？
