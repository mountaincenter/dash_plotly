01 / Hypothesis

仮説: ボラティリティ × 出来高で
シグナルの質を分離する

B4（逆張り）の仮説

乖離-8%超からの反発を狙うシグナル。

高ATR = 大きく動いている銘柄
→ 急落がパニック売りなら、反発も大きい
→ ATRが低い-8%乖離はジリ下げで反発力が弱い

高出来高 = 参加者が多い急落
→ パニック売りは出来高を伴う
→ 閑散下落は構造的で反発しない

予測: 高ATR + 高Vol → PF改善

B1/B3（順張り）の仮説

SMA上抜け・SMAタッチ反発のトレンドフォロー。

低ATR = 安定した値動き
→ 静かにSMAを上抜けるGCは「ダマシ」が少ない
→ ボラが高い時のGCは乱高下中のノイズ

出来高は順張りでは方向が不定
（静かな地合いでの安定上昇が良い場合も）

予測: 低ATR → PF改善

検証設計（過学習を防ぐ）

Train期間: 2014-03 ~ 2019-12（6年）→ 最適閾値を発見
Test期間: 2020-01 ~ 2026-02（6年）→ 同じ閾値で再現するか検証

Testで再現 = 構造的なエッジ（採用）
Testで消滅 = 過学習（不採用）
1変数ずつ追加し、差分を確認。複合効果も検証。

なぜ相乗するか

ATR%は「銘柄の通常ボラティリティが高い」ことを捉える（価格ベース）。
出来高比は「その日に参加者が増えている」ことを捉える（取引量ベース）。
高ATR + 高出来高 = 高ボラ銘柄に参加者が殺到 → パニック売りの典型パターン。
2変数は異なる情報源なので、組み合わせて精度が上がる。

B1/B3: 出来高は不採用

B1 高出来高: Train PF改善 → Test PF悪化（方向不一致）
B1 低出来高: Train PF悪化 → Test PF改善（方向が逆転）

B3 高出来高: 単独ではOOS通過（Vol≥1.0: Train 1.76→Test 1.98）。
しかしATR≤1.5 と組み合わせると N が減りすぎて効果消滅。
ATR単体で PF 2.0+が出ているので、出来高追加は不要。

なぜ低ATRが良いのか

B1(GC突破)もB3(SMAタッチ反発)もトレンドの継続に賭ける。
ATRが高い = 方向感がない乱高下。
SMAを上抜けてもすぐ下抜ける「ダマシ」が頻発。

ATR% ≤ 1.5 の銘柄は日次変動が終値の1.5%以内。
穏やかに上昇する銘柄のGCが「本物」になりやすい。
