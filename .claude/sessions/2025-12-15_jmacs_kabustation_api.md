### 2. kabu STATION API 自動売買への移行

#### 目的
- ルールをコードで**強制**する
- 感情的な取引を排除

#### 強制するルール
```python
Side: '1' (売り) 以外は実行しない  # ロング禁止
MarginTradeType: 3 (デイトレ) 固定  # 当日決済強制
同一銘柄ポジションチェック          # ナンピン禁止
14時に全ポジション自動決済
```

#### 環境
- **証券会社**: 三菱UFJ eスマート証券（口座開設済み、信用口座あり）
- **実行環境**: Windows（所有済み）
- **必要プラン**: Professionalプラン

#### Professionalプラン適用条件
| 条件 | 状況 |

## 技術メモ

### kabu STATION API

- **ベースURL**: `http://localhost:18080/kabusapi`
- **認証**: APIパスワードでトークン取得
- **注文エンドポイント**: `/sendorder`

### 信用新規売り（ショート）パラメータ例
```python
{
    'Side': '1',           # 1=売り, 2=買い
    'CashMargin': 2,       # 1=現物, 2=信用
    'MarginTradeType': 3,  # 1=制度信用, 2=長期, 3=デイトレ
    'Symbol': '5817',      # 銘柄コード
    'Exchange': 1,         # 1=東証
    'Qty': 100,            # 数量
    'Price': 0,            # 0=成行
}
```
