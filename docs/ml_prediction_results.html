analysis vs ML: アプローチの違い

analysis / analysis-custom

  • RSI帯（0-10, 10-20, ...）で分類
  • 曜日ファクターで分類
  • 時間帯（10:25, 前引, 14:45, 大引）で分類
  • 価格帯・時間帯の「区切り」で判断
  • 攻めの戦略: いつ入るか

ML予測

  • 34特徴量を総合的に評価
  • RSI、出来高、ボラ、市場環境を同時に考慮
  • 個別銘柄ごとに確率を算出
  • 「区切り」ではなく連続的な確率
  • 守りの戦略: 何を避けるか
